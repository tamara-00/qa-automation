# AI-Assisted QA Automation - PrestaShop Demo

## Netcetera x FINK Collaboration
This project was developed as part of a collaboration between **Netcetera** and the **Faculty of Computer Science & Engineering (FINKI)**, focused on applying AI in Quality Assurance and test automation workflows.

The project explores how AI tools can support automation testing by analyzing manual test cases and identifying suitable automation candidates.

## About the Project
This repository contains automation tests created for the **PrestaShop Demo** web application.

The implemented test scenarios are based on test cases and test charters from **Sprint 1** and **Sprint 2**, which were analyzed as input for identifying automation opportunities and maximizing test coverage.

Using AI-assisted workflows, selected functionalities were automated and compared against manually written automation tests.

## Project Workflow
The project includes:

- Analysis of Sprint 1 and Sprint 2 test cases
- Identification of automation candidates using AI tools
- Division of functionalities among team members
- Development of automation tests for selected features

For selected functionalities:
- 1 automation test written manually (without AI assistance)
- Minimum 2 automation tests created with AI assistance

## Comparison Criteria
Human-written and AI-assisted tests are compared based on:

- Efficiency
- Stability
- Maintainability
- Accuracy
- Code structure and readability

The goal is to evaluate the practical benefits and limitations of AI in real-world QA automation processes.

## Repository Content
This repository includes:
- Human-written automation tests
